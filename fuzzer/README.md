# Fuzzer

필요에 의해서 만든 간단한 퍼저입니다.

## 프로젝트 목적

개발 중 디버깅 및 오류발견을 용이하기 하기 위해, 여러 상황에 맞는 인자를 주는 퍼저를 작성한다.

## 목표

### 프로그램 구조

- [X] 간단한 퍼징
  현재는 인자를 주는 정도의 간단한 퍼징만 구현되었습니다.
- [ ] json값에 따른 인자값 생성
- [ ] json값에 따른 파일 생성 및 해당 값을 인자로 주기

### 기능 목표

- [ ] Json값에 따른 상세 퍼징
- [ ] 네트워크에 접속하는 퍼싱
- [ ] 임시 디렉토리를 생성해 인자로 사용할 파일 생성
- [ ] 스레드 사용으로 병렬처리
- [ ] 포크를 사용한 퍼징 가속화
- [ ] 로그 작성
- [ ] 여러 결과에 따른 오류 탐지
